Milestone Report

Summary of Completed Work

We planned out the overall project structure including the different classes and files we would need. We also set up a project repository that imports the glm and OpenGL libraries and uses CMake for compilation. Next, we made a simple renderer that orthographically projects the particles onto a 2d plane. We implemented most of the features for our sequential simulator and made some scene files to test on. In our current implementation, we are able to load a scene file, step the sequential simulator on it, and draw the current state of the particles. We have begun to plan out our parallel simulator.
